# hexagonpy
Python script to convert an image into a grid of hexagon tiles

## requirements
Requires numpy and pillow libraries to be installed:
```
pip install numpy
pip install pillow
```


# Usage

```bash
python image.py Lenna.png 5
python image.py A_sunflower.jpg 20
```
